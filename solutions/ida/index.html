IDA · Intelligent Document Analyzer

문서 데이터를 정량 근거로 분석

IDA는 MetaInsight의 제품 경험을 기반으로, 구조화된 문서·레코드 데이터에서 키워드 클러스터와 시계열 시그널을 찾고 분석 결과를 인사이트로 정리하는 문서 분석 AI 솔루션입니다.

Product Scope

문서·레코드 분석을 위한 핵심 기능

IDA는 텍스트와 날짜 축이 정리된 문서·레코드 데이터를 기준으로 분석을 시작합니다. 업로드, 컬럼 매핑, 클러스터링, 정량 평가, 인사이트 생성까지 이어지는 흐름을 하나의 화면 경험으로 제공합니다.

문서·레코드 업로드

CSV, Excel, Parquet 파일을 업로드하고 텍스트, 날짜, 금액, 그룹, 식별자 컬럼을 화면에서 확인하며 분석 조건을 구성합니다.

키워드 클러스터링

비슷한 키워드와 문서 항목을 자동으로 묶고, 각 클러스터가 시간에 따라 어떻게 변화하는지 정량적으로 비교합니다.

정량 근거 기반 인사이트

C1 장기 트렌드, C2 모멘텀 유지, C3 최근 가속 기준을 바탕으로 등급과 점수를 산출하고 자연어 인사이트를 제공합니다.

Analysis Workflow

업로드부터 인사이트까지 이어지는 운용 흐름

사용자는 복잡한 코드 없이 분석 파일을 올리고, 자동 감지된 컬럼을 검토한 뒤 프리셋을 선택해 분석을 실행합니다. 결과는 요약 지표, 차트, 항목 상세, 자연어 인사이트로 나누어 확인합니다.

01

Upload

분석할 문서·레코드 데이터를 업로드하고 지원 형식을 확인합니다.

02

Map Columns

텍스트와 날짜 컬럼을 중심으로 분석에 필요한 필드를 매핑합니다.

03

Evaluate

키워드 군집을 만들고 트렌드 강도, 모멘텀, 최근 가속을 평가합니다.

04

Review Insights

요약 지표, 상세 항목, AI 또는 템플릿 기반 인사이트를 검토합니다.

정량 평가 엔진

단순 키워드 빈도보다 중요한 것은 지속성, 현재성, 변화 속도입니다. IDA는 각 클러스터를 독립 기준으로 평가해 설명 가능한 점수와 등급으로 정리합니다.

C1

장기 트렌드

시간이 지나며 꾸준히 증가하는 클러스터인지 확인합니다.

C2

모멘텀 유지

최근 구간의 활동이 기준선보다 높은 상태를 유지하는지 봅니다.

C3

최근 가속

현재 시점에 가까워질수록 격차가 더 커지는지 평가합니다.

결과 화면에서는 이 기준을 통과한 수, 종합 점수, 관련 키워드, 레코드 수를 함께 보여주어 분석자가 근거를 추적할 수 있게 합니다.

Product Screens

선택해서 보는 실제 화면

의사결정에 필요한 장면을 선별했습니다. 왼쪽에서 관심 기능을 선택하면 오른쪽에서 실제 화면과 핵심 의미를 바로 확인할 수 있습니다.

결과 요약

레코드, 그룹, 감지된 시그널, 전체 클러스터를 한 화면에서 확인

Operation & Expansion

분석 엔진을 제품으로 운용하기 위한 기반

MetaInsight는 분석 엔진만이 아니라 계정, 프로젝트, 결과 저장, 내보내기, 모바일 대응 화면까지 포함한 제품형 구조를 갖추고 있습니다. IDA는 이 기반 위에서 기업 문서·규제·과제 데이터 분석 영역으로 확장됩니다.

SaaS 운용 구조

JWT 인증, 프로젝트 단위 관리, 사용량 제한 등 SaaS 제품 운용에 필요한 기본 구조를 갖춥니다.

도메인 확장

R&D 과제, 특허, 리테일, 금융 리스크처럼 텍스트와 날짜 축이 있는 데이터로 분석 대상을 확장할 수 있습니다.

반복 탐색

국가, 분야, 기간, 프리셋 등 조건을 조정하며 같은 데이터에서 다른 관점의 시그널을 확인합니다.

문서·레코드에서 실행 가능한 인사이트를 찾으세요.

IDA는 업로드, 클러스터링, 정량 평가, 인사이트 생성을 하나의 분석 흐름으로 제공합니다.
